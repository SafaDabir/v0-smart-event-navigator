# AI Crowd Flow Optimizer 🚀

A smart web application designed to improve physical event experiences by optimizing crowd flow, reducing wait times, and suggesting efficient routes.

## ✨ Features
- Real-time crowd level indicators (Low / Medium / High)
- Estimated wait times for each zone
- AI-based route suggestion for minimal congestion
- Clean and responsive dark UI dashboard

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- Vercel (Deployment)

## 🔗 Live Demo
👉 https://v0-smart-event-navigator.vercel.app

## 📌 Problem Statement
Design a solution that improves physical event experiences by managing crowd movement, reducing waiting times, and enabling better coordination at large-scale venues

Built as part of a hackathon challenge.
